This is a C programming banking system project called Digital Ledger Bank. It allows users to manage customer accounts, perform transactions, and calculate interest for savings and fixed accounts. This project was developed as a mini project for learning purposes.

Features

Create new customer accounts with personal details and initial deposit.

Update existing account information (address and phone number).

Deposit and withdraw money from accounts.

View account details by account number or customer name.

Delete customer accounts.

List all customer accounts.

Interest calculation for saving and fixed accounts.

Password-protected login for access.

Technologies Used

C programming language

File handling for storing customer records

Basic console interface for interaction

How to Run

Open the project in a C IDE (like Dev-C++, Code::Blocks, or Visual Studio).

Compile the main.c file.

Run the executable in the console.

Enter the password digital to access the main menu.

Use the menu to manage accounts and transactions.

Notes

Fixed accounts (fixed1, fixed2, fixed3) do not allow deposits or withdrawals until maturity.

This project is for educational purposes and learning C programming.
